  • Punch Clock

    • Punch Clock is a fully functioning time keeping app me and my friend worked on. The app allows you to make "categories" in which the user can then enter time entries into. An example might be keeping track of your hours or working towards a goal of a certain amount of hours per week. I learned a ton about Java and Android programming with this application.

    FormatTimeUtil.java is a simple Java class I made for formatting numbers into a one of two different formats for displaying time. It has two methods, one for converting a long value into the format of "hh:mm:ss" and another for "00d00h00m". This class was used to format the display of the total time entered into a category and various instances of timers.

    This is the home screen for Punch Clock. Here, you can create a category and begin your time keeping. The total time committed to each category is displayed, the favorites icon is listed, and you are given an array of buttons that grant stopwatch functionality. Clicking on a category card will bring you to the detail page for that category.

    This is the detail screen. Night Mode is activated. It looked good. you can list a bunch of different times here.
